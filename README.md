# file-find
file-find — find files by name/glob pattern with structured JSON output. Size, mode, modification time per file.

Part of [supercli](https://github.com/javimosch/supercli) ecosystem.

```bash
curl -LO https://github.com/javimosch/file-find/releases/latest/download/file-find-linux-amd64
chmod +x file-find-linux-amd64; sudo mv file-find-linux-amd64 /usr/local/bin/file-find
```
